#url Shortener a Narrow Scope Tool

## Installation

Clone the codebase to your local environment.

```shell
git clone git@github.com:reactivepixel/urlShortner.git
```

Install Packages

```
npm install
```

## Testing the Tool

Make certain you have mocha installed globally

```
npm install -g mocha
```

Run the Mocha Tests

```
mocha
```
